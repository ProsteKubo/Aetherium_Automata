\chapter{Contents of the Included Storage Media}
\label{app:media}

% TODO: fill in final directory listing once all deliverables are collected.

The included storage medium contains the following items:

\begin{description}
  \item[\texttt{src/}] Full source code of the Aetherium Automata platform:
    \begin{itemize}
      \item \texttt{src/engine/} --- C++17 runtime engine and platform targets
      \item \texttt{src/ide/} --- Electron IDE (TypeScript/React)
      \item \texttt{src/gateway/} --- Elixir/Phoenix gateway
      \item \texttt{src/server/} --- Elixir server
    \end{itemize}
  \item[\texttt{example/}] Showcase automata catalog:
    \begin{itemize}
      \item \texttt{example/automata/showcase/} --- 39 YAML automata definitions
      \item \texttt{example/ide\_demo\_projects/} --- IDE demo project files (\texttt{.aeth})
    \end{itemize}
  \item[\texttt{thesis/}] \LaTeX{} source of this technical report and compiled PDF.
\end{description}
